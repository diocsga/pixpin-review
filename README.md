## 跟 Snipaste 比怎么样？

这是很多人会问的问题，因为 Snipaste 也是截图贴图类工具里的老牌选手。

Snipaste 做得精，截图和贴图体验非常流畅，有些用户反馈它的响应速度比 PixPin 稍快一点。但功能比较集中，没有 OCR、没有长截图、没有动图录制。

PixPin 的定位基本上是「Snipaste + OCR + 长截图 + GIF 录制」的合集版本。如果你同时需要这几个功能，用 PixPin 可以省掉装好几个软件的麻烦；如果你只需要截图贴图，Snipaste 也完全够用。
